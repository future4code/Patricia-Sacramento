Fixando os conceitos essenciais:

1. PROPS:
Props são propriedades passadas do componente pai para os componentes filhos. 
São parâmetros apenas para leitura do componente filho, ou seja,
o componente filho não pode alterar o valor da props. 

2. STATE:
State é um objeto que guarda propriedades que serão transformadas ao longo do
tempo. 

3. COMPONENTE: 
Componente é um bloco de código independente e reutilizável.

4. DIFERENÇA ENTRE COMPONENTE DE CLASSE E FUNCIONAL:
O componente funcional é o mesmo que "componente de função", ou seja, é uma 
função JavaScript. O componente de classe é o componente que é definido
por meio de classes, não de funções. O componente funcional preocupa-se com
a apresentação dos dados na aplicação, o de classe lidam com as partes lógicas
da aplicação, manupulam eventos e herdam os ciclos de vida do React. 

5. HOOKS:
Os Hooks são grupos de funções que dão capacidades extras aos componentes
funcionais sem a necessidade de escrever uma classe. 

6. USESTATE:
O useState é um Hook que permite adicionar um state a um componente funcional
e atualizá-lo por meio de uma função.

7. USEEFFECT:
O useEffect é um Hook que permite a execução das funcionalidades dos métodos
de ciclo de vida em componentes funcionais.
